## แผนการจัดการเรียนรู้รายสัปดาห์

### สัปดาห์ที่ 1-2: หน่วยที่ 1 - ความรู้พื้นฐานเกี่ยวกับระบบไอโอทีและอุปกรณ์
**จำนวนคาบ:** 12 คาบ (ทฤษฎี 2 คาบ, ปฏิบัติ 10 คาบ)

#### สัปดาห์ที่ 1

**คาบที่ 1 (ทฤษฎี - 50 นาที)**
- **หัวข้อ:** ความหมายและความสำคัญของระบบไอโอที
- **กิจกรรม:**
  - บรรยายเรื่องความหมายของ IoT
  - อภิปรายเกี่ยวกับการประยุกต์ใช้ IoT ในชีวิตประจำวัน
  - ดูวิดีโอตัวอย่างการใช้งาน IoT
- **สื่อการสอน:** PowerPoint, วิดีโอ, ตัวอย่างอุปกรณ์ IoT
- **การประเมิน:** การสังเกตการมีส่วนร่วมในการอภิปราย

**คาบที่ 2-3 (ปฏิบัติ - 100 นาที)**
- **หัวข้อ:** สถาปัตยกรรมระบบไอโอที
- **กิจกรรม:**
  - ศึกษาสถาปัตยกรรม IoT (3-Layer, 4-Layer, 5-Layer)
  - วิเคราะห์องค์ประกอบของระบบ IoT
  - ฝึกวาดแผนภาพสถาปัตยกรรม IoT
- **สื่อการสอน:** เอกสารประกอบ, แผนภาพ, ตัวอย่างระบบ
- **การประเมิน:** ตรวจแผนภาพสถาปัตยกรรม

**คาบที่ 4-5 (ปฏิบัติ - 100 นาที)**
- **หัวข้อ:** องค์ประกอบของระบบไอโอที
- **กิจกรรม:**
  - ศึกษาอุปกรณ์ Sensors ชนิดต่างๆ
  - ศึกษาอุปกรณ์ Actuators
  - ศึกษาอุปกรณ์ Gateways และ Cloud Platform
  - ฝึกจำแนกประเภทอุปกรณ์ IoT
- **สื่อการสอน:** อุปกรณ์จริง, เอกสารประกอบ, แผนภาพ
- **การประเมิน:** แบบทดสอบย่อย

#### สัปดาห์ที่ 2

**คาบที่ 6 (ทฤษฎี - 50 นาที)**
- **หัวข้อ:** โพรโทคอลการสื่อสารพื้นฐาน
- **กิจกรรม:**
  - บรรยายเรื่อง HTTP, MQTT, CoAP
  - เปรียบเทียบโพรโทคอลต่างๆ
  - อภิปรายการเลือกใช้โพรโทคอล
- **สื่อการสอน:** PowerPoint, ตัวอย่างการใช้งาน
- **การประเมิน:** การสังเกตการมีส่วนร่วม

**คาบที่ 7-8 (ปฏิบัติ - 100 นาที)**
- **หัวข้อ:** การวิเคราะห์ความต้องการใช้งาน
- **กิจกรรม:**
  - ศึกษาวิธีการวิเคราะห์ความต้องการ
  - ฝึกวิเคราะห์กรณีศึกษา
  - จัดทำรายงานการวิเคราะห์ความต้องการ
- **สื่อการสอน:** กรณีศึกษา, เอกสารประกอบ
- **การประเมิน:** รายงานการวิเคราะห์ความต้องการ

**คาบที่ 9-10 (ปฏิบัติ - 100 นาที)**
- **หัวข้อ:** ศึกษาค้นคว้าและนำเสนอเกี่ยวกับอุปกรณ์ไอโอที
- **กิจกรรม:**
  - แบ่งกลุ่มศึกษาค้นคว้าอุปกรณ์ IoT
  - จัดทำรายงานและนำเสนอ
  - อภิปรายและแลกเปลี่ยนเรียนรู้
- **สื่อการสอน:** อินเทอร์เน็ต, เอกสารประกอบ
- **การประเมิน:** รายงานและคะแนนการนำเสนอ

**คาบที่ 11-12 (ปฏิบัติ - 100 นาที)**
- **หัวข้อ:** สรุปและทดสอบหน่วยที่ 1
- **กิจกรรม:**
  - สรุปเนื้อหาหน่วยที่ 1
  - ทำแบบทดสอบความรู้พื้นฐาน
  - ตรวจสอบความเข้าใจ
- **สื่อการสอน:** แบบทดสอบ
- **การประเมิน:** แบบทดสอบความรู้พื้นฐาน (20 คะแนน)

## การจัดการทรัพยากรและสื่อการสอน

### อุปกรณ์และเครื่องมือที่ต้องเตรียม

#### อุปกรณ์ไอโอที
1. **Smart Switch** - จำนวน 10-15 ชิ้น
2. **Sensors** (Temperature, Humidity, Motion, Light) - จำนวน 20-30 ชิ้น
3. **Actuators** (Relay, Motor) - จำนวน 10-15 ชิ้น
4. **IR Remote Control** - จำนวน 5-10 ชิ้น
5. **433 MHz Modules** (Transmitter, Receiver) - จำนวน 10-15 ชิ้น
6. **Bluetooth Modules** - จำนวน 10-15 ชิ้น
7. **ZigBee Modules** - จำนวน 10-15 ชิ้น
8. **Smart CCTV** - จำนวน 3-5 ตัว
9. **Voice Assistant Devices** (Google Home, Amazon Echo) - จำนวน 2-3 ตัว

#### อุปกรณ์คอมพิวเตอร์และเครือข่าย
1. **คอมพิวเตอร์** - จำนวน 15-20 เครื่อง
2. **Raspberry Pi** - จำนวน 5-10 ตัว (สำหรับ Home Assistant)
3. **Router/WiFi Access Point** - จำนวน 2-3 ตัว
4. **สมาร์ทโฟน/แท็บเล็ต** - จำนวน 15-20 เครื่อง

#### ซอฟต์แวร์
1. **IDE** (Visual Studio Code, Arduino IDE, PlatformIO)
2. **UML Software** (Draw.io, StarUML)
3. **Home Assistant**
4. **แอปพลิเคชันจัดการ IoT** (Smart Life, Tuya, Home Assistant App)
5. **Development Frameworks** (React, Flutter, Node.js)

#### เครื่องมือและอุปกรณ์เสริม
1. **เครื่องมือช่าง** (ไขควง, คีม, มัลติมิเตอร์)
2. **สายไฟและอุปกรณ์ไฟฟ้า**
3. **Breadboard และ Jumper Wires**
4. **Power Supply**

### การจัดเตรียมห้องเรียน
- **ห้องปฏิบัติการคอมพิวเตอร์** - สำหรับการพัฒนาแอปพลิเคชัน

- **พื้นที่ทดสอบ** - สำหรับการติดตั้งระบบ Smart Home แบบจำลอง

- **วิธีแก้ไข:**
  - ตรวจสอบ API Documentation
  - ใช้ Debugging Tools
  - ทดสอบทีละส่วน

#### 4. ปัญหาอุปกรณ์ไม่เพียงพอ
- **สาเหตุ:** จำนวนนักเรียนมาก, งบประมาณจำกัด
- **วิธีแก้ไข:**
  - แบ่งกลุ่มทำงาน
  - ใช้ Simulation Software
  - ขอสนับสนุนจากหน่วยงาน

### ข้อควรระวังด้านความปลอดภัย
1. **ความปลอดภัยทางไฟฟ้า**
   - ตรวจสอบแรงดันไฟฟ้าก่อนใช้งาน
   - ใช้เครื่องมือป้องกันไฟฟ้า
   - ปิดสวิตช์ก่อนติดตั้ง

2. **ความปลอดภัยทางไซเบอร์**
   - ตั้งรหัสผ่านที่แข็งแกร่ง
   - อัพเดทเฟิร์มแวร์เป็นประจำ
   - ใช้เครือข่ายแยกสำหรับ IoT

3. **ความปลอดภัยของข้อมูล**
   - ไม่เปิดเผยข้อมูลส่วนตัว
   - ใช้การเข้ารหัสข้อมูล
   - ตรวจสอบสิทธิ์การเข้าถึง

## การติดตามและประเมินผลการจัดการเรียนรู้

### การติดตามความก้าวหน้า
1. **การบันทึกการเข้าเรียน** - ติดตามการเข้าเรียนของนักเรียน
2. **การตรวจงาน** - ตรวจงานที่มอบหมายเป็นระยะ
3. **การสังเกตการณ์** - สังเกตการมีส่วนร่วมและการปฏิบัติงาน
4. **การสอบถาม** - สอบถามความเข้าใจและปัญหา

### การปรับปรุงการสอน
1. **การประเมินผลการสอน** - ประเมินผลการสอนจากนักเรียน
2. **การวิเคราะห์ผลการเรียน** - วิเคราะห์คะแนนและจุดอ่อน
3. **การปรับแผนการสอน** - ปรับแผนตามผลการประเมิน
4. **การพัฒนาสื่อการสอน** - พัฒนาสื่อให้เหมาะสม
