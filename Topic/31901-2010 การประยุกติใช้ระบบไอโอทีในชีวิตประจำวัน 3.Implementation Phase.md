# Implementation Phase - การประยุกติใช้ระบบไอโอทีในชีวิตประจำวัน

## 1. ลักษณะรายวิชา

**รหัสวิชา:** 31901-2010  
**ชื่อวิชา:** การประยุกติใช้ระบบไอโอทีในชีวิตประจำวัน  
**ชื่อวิชาภาษาอังกฤษ:** IoT in daily life  
**หน่วยกิต:** 3 (1-4-3)  
**จำนวนชั่วโมง:** 75 ชั่วโมง (ทฤษฎี 15 ชั่วโมง, ปฏิบัติ 60 ชั่วโมง)

**คำอธิบายรายวิชา:**  
ศึกษาและปฏิบัติเกี่ยวกับการติดตั้งและใช้งานอุปกรณ์ไอโอทีภายในบ้าน สวิตซ์ไอโอที อุปกรณ์ควบคุมรีโมทอินฟราเรด การเชื่อมต่อด้วยสัญญาณวิทยุ (433 MHz) ในการสื่อสาร (Bluetooth ZigBee) อุปกรณ์รักษาความปลอดภัยภายในบ้าน การติดตั้ง Home Assistant ระบบสั่งงานด้วยเสียง (Voice Assistant) โพรโทคอล Matter จัดทำซอฟต์แวร์ตาม UML Modeling ด้านซอฟต์แวร์เพื่ออินเตอร์เน็ตของสรรพสิ่งแบบหนึ่งต่อหนึ่ง สร้างส่วนติดต่อผู้ใช้ (User Interface) และเชื่อมต่อกับซอฟต์แวร์ภายนอก (APIs) ด้านซอฟต์แวร์เพื่ออินเตอร์เน็ตของสรรพสิ่ง ทดสอบโปรแกรมย่อยซอฟต์แวร์ ด้านซอฟต์แวร์เพื่ออินเตอร์เน็ตสรรพสิ่ง การประยุกต์ใช้งานระบบไอโอทีในชีวิตประจำวัน จัดทำคู่มือการใช้งานโปรแกรมระบบไอโอที

**อ้างอิงมาตรฐาน:**  
มาตรฐานอาชีพ สถาบันคุณวุฒิวิชาชีพ อาชีพนักพัฒนาซอฟต์แวร์เพื่ออินเตอร์เน็ตของสรรพสิ่ง ระดับ 4

**ผลลัพธ์การเรียนรู้ระดับรายวิชา:**  
ติดตั้งอุปกรณ์ไอโอทีและบริหารจัดการให้อยู่ในระบบนิเวศน์เดียวกันตามหลักการ ด้วยความละเอียดรอบคอบ รับผิดชอบ การสื่อสาร การคิดเชิงนวัตกรรมและการทำงานเป็นทีม

**จุดประสงค์รายวิชา:**  
1. เข้าใจเกี่ยวกับอุปกรณ์ไอโอที และวิธีการติดตั้งเพื่อใช้งาน  
2. มีทักษะในการติดตั้งอุปกรณ์ไอโอทีเพื่อใช้งานในชีวิตประจำวัน  
3. มีเจตคติและกิจนิสัยที่ดีในการปฏิบัติงานด้วยความละเอียดรอบคอบ รับผิดชอบ การสื่อสารการคิดเชิงนวัตกรรมและการทำงานเป็นทีม  
4. มีความสามารถประยุกต์ใช้อุปกรณ์ไอโอทีเพื่อใช้งานในชีวิตประจำวัน

**สมรรถนะรายวิชา:**  
1. ประมวลความรู้เกี่ยวกับอุปกรณ์ไอโอที และวิธีการติดตั้งเพื่อใช้งาน  
2. เลือกและติดตั้งแพลตฟอร์มให้เหมาะสมกับอุปกรณ์ไอโอที  
3. บริหารจัดการอุปกรณ์เพื่อให้ใช้งานได้ในระบบนิเวศเดียวกัน  
4. ประยุกต์ใช้อุปกรณ์ไอโอทีเพื่อใช้งานในชีวิตประจำวัน

### รายละเอียดหน่วยการเรียนรู้

**หน่วยที่ 1: ความรู้พื้นฐานเกี่ยวกับระบบไอโอทีและอุปกรณ์**  
- จุดประสงค์: อธิบายความหมายและหลักการทำงานของระบบไอโอทีได้ ระบุประเภทและคุณสมบัติของอุปกรณ์ไอโอทีได้ อธิบายสถาปัตยกรรมและองค์ประกอบของระบบไอโอทีได้ วิเคราะห์ความต้องการในการใช้งานระบบไอโอทีในชีวิตประจำวันได้  
- สาระการเรียนรู้: ความหมายและความสำคัญของระบบไอโอที สถาปัตยกรรมระบบไอโอที องค์ประกอบของระบบไอโอที ประเภทอุปกรณ์ไอโอทีและการใช้งาน โพรโทคอลการสื่อสารพื้นฐาน การวิเคราะห์ความต้องการใช้งาน

**หน่วยที่ 2: การติดตั้งและจัดการอุปกรณ์ไอโอทีพื้นฐาน**  
- จุดประสงค์: ติดตั้งอุปกรณ์ไอโอทีพื้นฐานภายในบ้านได้ ตั้งค่าและเชื่อมต่อสวิตช์ไอโอทีได้ ใช้งานอุปกรณ์ควบคุมรีโมทอินฟราเรดได้ บริหารจัดการอุปกรณ์ไอโอทีผ่านแอปพลิเคชันได้  
- สาระการเรียนรู้: การติดตั้งสวิตช์ไอโอที การตั้งค่าและเชื่อมต่ออุปกรณ์ไอโอที การใช้งานรีโมทอินฟราเรด การเชื่อมต่อด้วยสัญญาณวิทยุ 433 MHz การจัดการอุปกรณ์ผ่านแอปพลิเคชันมือถือ การแก้ไขปัญหาการเชื่อมต่อ

**หน่วยที่ 3: โพรโทคอลการสื่อสารและระบบนิเวศน์ไอโอที**  
- จุดประสงค์: อธิบายโพรโทคอลการสื่อสารไอโอทีได้ ใช้งานโพรโทคอล Bluetooth และ ZigBee ได้ ติดตั้งและใช้งาน Home Assistant ได้ บริหารจัดการอุปกรณ์ให้อยู่ในระบบนิเวศน์เดียวกันได้  
- สาระการเรียนรู้: โพรโทคอลการสื่อสารไร้สาย โพรโทคอล Bluetooth และการใช้งาน โพรโทคอล ZigBee และการใช้งาน การติดตั้ง Home Assistant การเชื่อมต่ออุปกรณ์เข้ากับ Home Assistant การสร้าง Automation และ Scene โพรโทคอล Matter และการใช้งาน

**หน่วยที่ 4: ระบบรักษาความปลอดภัยและระบบสั่งงานด้วยเสียง**  
- จุดประสงค์: ติดตั้งและใช้งานอุปกรณ์รักษาความปลอดภัยได้ ตั้งค่าระบบแจ้งเตือนได้ ใช้งานระบบสั่งงานด้วยเสียงได้ เชื่อมต่อ Voice Assistant กับอุปกรณ์ไอโอทีได้  
- สาระการเรียนรู้: อุปกรณ์รักษาความปลอดภัยไอโอที ระบบกล้องวงจรปิดอัจฉริยะ ระบบเซ็นเซอร์ตรวจจับการเคลื่อนไหว ระบบแจ้งเตือน ระบบสั่งงานด้วยเสียง การเชื่อมต่อ Google Assistant / Amazon Alexa การสร้างคำสั่งเสียง

**หน่วยที่ 5: การพัฒนาซอฟต์แวร์ไอโอทีด้วย UML และการสร้าง User Interface**  
- จุดประสงค์: ออกแบบซอฟต์แวร์ไอโอทีด้วย UML Modeling ได้ พัฒนาซอฟต์แวร์ไอโอทีแบบหนึ่งต่อหนึ่งได้ สร้างส่วนติดต่อผู้ใช้ได้ เชื่อมต่อกับซอฟต์แวร์ภายนอก (APIs) ได้  
- สาระการเรียนรู้: การออกแบบด้วย UML การพัฒนาแอปพลิเคชันไอโอที การพัฒนาแบบหนึ่งต่อหนึ่ง การสร้าง User Interface การเชื่อมต่อ API การจัดการข้อมูล การทดสอบโปรแกรมย่อย

**หน่วยที่ 6: การประยุกต์ใช้ระบบไอโอทีในชีวิตประจำวันและการจัดทำเอกสาร**  
- จุดประสงค์: ประยุกต์ใช้ระบบไอโอทีในชีวิตประจำวันได้ จัดทำคู่มือการใช้งานโปรแกรมระบบไอโอทีได้ นำเสนอผลงานได้ ประเมินและปรับปรุงระบบได้  
- สาระการเรียนรู้: การประยุกต์ใช้ระบบไอโอทีในสถานการณ์จริง การออกแบบระบบไอโอทีสำหรับบ้านอัจฉริยะ การจัดทำคู่มือการใช้งาน การจัดทำเอกสารเทคนิค การนำเสนอผลงาน การประเมินผลและปรับปรุงระบบ
